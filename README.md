## Setup

```powershell
# installing dependencies
pnpm install

# building the vite project
pnpm build

# previewing the built vite project
pnpm preview

# or

# installing dependencies
npm install
# building the vite project
npm run build

# previewing the built vite project
npm run preview
```

## Dev Setup

```powershell
# installing dependencies
pnpm install
# running vite dev server
pnpm dev
# formatting src files using prettier
pnpm format

# or

# installing dependencies
npm install
# running vite dev server
npm run dev
# formatting src files using prettier
npm run format
```
